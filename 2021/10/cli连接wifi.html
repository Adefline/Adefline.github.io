CLI连接wifi

2021-10-23 ::

By: NickName

linux系在命令行连接wifi的教程根据环境不同有很大区别，本机使用Ddebian11，安装ip命令。

使用ip或ls /sys/class/net查看无线网卡。

本机网卡为wlp3s0.

使用sudo ip link set wlp3s0 up打开网卡

使用sudo iw dev wlp3s0 scan |grep -E 'SSID|signal|WPA'查看网卡扫描的无线网络

选择想要连接的wifi,记住他的ssid和密码

编辑或添加/etc/wpa_supplicant/wpa_supplicant.conf文件

ctrl_interface=/var/run/wpa_supplicant

ctrl_interface_group=0

ap_scan=1

network={
        ssid="[wifi名]"
        psk="[密码]"
        priority=1
}

sudo wpa_supplicant -i wlp3s0 -c /etc/wpa_supplicant/wpa_supplicant.conf 连接wifi,后台连接或在一个tty中连接后切换tty。

可使用ip命令查看网卡连接状态。 连接后配置DHCP获取ip

使用sudo dhclient获取ip
